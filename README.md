これはなに？
---

csv形式のファイルをアップロードし，web上でグラフを簡単に閲覧するためのサービスです

開発を始める
---

初回のときは次のコマンドを実行してください．

- bundle install
- npm install

次のコマンドを実行してください．

- bundle exec foreman start
- npm run dev
